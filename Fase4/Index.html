Implementación Framework Bootstrap
Con el bootstrap podemos optimizar la interfaz tanto en navegadores como para todos los aplicativos en pc, laptop o móviles como smartphone o tablet sin perder la navegabilidad.
Para comprender un poco mejor visitar y/o accedar al aplicativo web ARWEB (2020) quien expone "Bootstrap, es un framework originalmente creado por Twitter, que permite crear interfaces web con CSS y JavaScript, cuya particularidad es la de adaptar la interfaz del sitio web al tamaño del dispositivo en que se visualice. Es decir, el sitio web se adapta automáticamente al tamaño de una PC, una Tablet u otro dispositivo"
