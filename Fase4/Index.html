FASE 4 - DISEÑO Y CONSTRUCCION III

Introducción

Fase No. 4:
La tercera fase de nuestro diseño del sitio web contempla evolución de las fases anteriores “Fase 1 y Fase 2”. se tomarán los elementos desarrollados anteriormente dentro de los cuales se implementarán cada uno de los nuevos elementos indicados dentro de la actividad de la presente actividad dentro de los cuales son.

Css3 : Las siglas CSS corresponden a Cascading Style Sheets; Podría definirse como un tipo de lenguaje que permite definir y crear la presentación de un documento ya estructurado y escrito en un lenguaje de marcado como puede ser HTML. Es decir, permite generar el diseño visual de páginas web e interfaces de usuario. Esta herramienta ha evolucionado con el tiempo y actualmente se encuentra en su tercera versión.

Grid Css : CSS Grid es un modelo de maquetación CSS en base a una rejilla, similar pero de manera mas evolucionada que tablas, este componente se implementa directamente desde código CSS para que los diseñadores pudieran posicionar los elementos en una distribución de filas y columnas

Iframe :Esta etiqueta es una línea de código de programación que permite colocar un elemento HTML dentro de otro elemento HTML principal.

Propósito de Actividad

El propósito de la Actividad conlleva la compresión e implementación de estilos para Etiquetas desde código css con lo cual se pretende una personalización completa de nuestro sitio iniciado anteriormente en las Fases 1 y 2. Para lo cual se requiere llegar a una comprensión del lenguaje css e identificación de elementos básicos utilizados en css cuya implementación logra un estilo en las etiquetas que nuestro sitio web.
